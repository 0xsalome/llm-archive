# Contributing

記事の追加や修正は、ブランチを作成してPull Requestで提案してください。

記事は `src/content/docs/archive/` に置き、軽量frontmatterを付けます。詳しい項目は [frontmatterガイド](docs/frontmatter-guide.md) を参照してください。

出典、不確かさ、AI関与、更新理由はできるだけ明示します。LLMによるリサーチ、構成、本文執筆、要約、言い換え、整理、レビュー補助を含む場合は、frontmatterまたは本文中にその関与を記録してください。

公開前には [公開前チェックリスト](docs/publishing-checklist.md) を確認します。記事は断定しすぎず、必要に応じて結論後に3行程度の「次に残る問い」や保留を残してください。
